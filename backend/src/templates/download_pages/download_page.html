  • Download the executable file for Linux
  • Open a terminal and navigate to the directory containing the downloaded file
  • Move the file accroding to your needs. (Optional)
  • Navigate to the parent folder of the downloaded file in terminal
  • Run the following command to make the file executable: chmod +x RemoteDesktop
  • Run the application with the following command: ./RemoteDesktop
