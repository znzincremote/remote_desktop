How to Use RemoteDesktop ?

Step 1: Download and Run App

To get started, simply download and run the Remote Desktop app. After launching the app, you'll be able to proceed to the next steps for connecting to remote systems securely.

Step 2: Authorize User

To get started, simply register using the desktop app. After registering, log in to the app to authorize your access. This is the first step towards connecting securely.

Step 3: Enter Remote Address

Your local system (your system) needs to connect to the remote system (the system you want to access). Simply enter the Remote System ID in the input field and click the "Start Connection" button.

Step 4: Accept or Reject Connection

Once you click "Start Connection", a message will be sent to the remote system asking for permission to connect. If the remote user accepts, your local system will be granted access to the remote system.
